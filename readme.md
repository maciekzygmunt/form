# Basic form for HexOcean

## How to run the project:

1. Clone this repo
2. Make `.env` file in root folder:

```env
VITE_BASE_API=https://umzzcc503l.execute-api.us-west-2.amazonaws.com/dishes
```

3. Run `npm i` and `npm run dev`
